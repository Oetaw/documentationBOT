Discord Bot Documentation

A comprehensive guide for understanding and using the Discord bot commands.

Introduction

This documentation provides an overview of a custom Discord bot developed using the discord.py library. The bot includes a variety of commands for server management, message handling, role and channel synchronization, and more.

Additionally, the bot leverages several helper functions and external modules for tasks such as AI-powered renaming, webhook handling, and link management.

Setup & Dependencies

To run this bot, ensure you have the following installed:

  • Python 3.8 or later
  • discord.py library
  • Modules: requests, asyncio, jellyfish
  • Custom modules: ai (for aichat), bypass, make_paste, and linkvertise

Update your bot token in the bot.run() call at the bottom of the code before running the bot.

Helper Functions

strip_emotes(text)

Removes Discord custom emoji tags and Unicode emojis from a given string.

discordify(text)

Converts text to a standardized format by lowercasing, replacing spaces with hyphens, and removing apostrophes. Useful for channel name comparisons.

search(name)

Searches the final.json file for an entry with the matching name and returns the entry if found.

switch(command, argument, embed)

Applies various formatting commands to a Discord embed. Supported commands include:

  • title: Sets the embed title.
  • description: Sets the embed description.
  • color: Sets the embed color (hex format expected).
  • image: Adds an image to the embed.
  • thumbnail: Adds a thumbnail image.
  • footer: Sets the footer text.
  • field: Adds a field (expects input in the format Field Name | Field Value).

Bot Commands

!test

Test Command

A simple command that prints the guild's icon URL to the console and sends a test message in the channel.

Example: !test
Expected Output: The bot sends a message test in the channel and logs the guild's icon URL in the console.

!scan

Scan Command

Scans the last 30 messages in specified text channels and checks for any words that match entries in the final.json file. If a match is found, the original message is deleted and a formatted notification is sent via a webhook.

Example: !scan
Expected Output: For any matched entry, a message like @here [Name] - [URL] is sent in the channel.

!send

Send Message / Embed Command

Sends a regular message or an embed message depending on the provided arguments.

Usage: !send [embed] <parameters> or !send <message>

  • For a plain message: !send Hello world!
  • For an embed message:
    !send embed /title Welcome /description "This is an embed message." /color FF5733
Example (Plain Message): !send Hello everyone!
Example (Embed): !send embed /title Greetings /description "Welcome to our server!" /footer "Enjoy your stay."
Expected Output (Plain Message): The bot sends "Hello everyone!" in the channel.
Expected Output (Embed): An embed is posted with the specified title, description, and footer.

!sync

Sync Command

Exports the serverâ€™s structure (roles, categories, channels, permission overwrites, and recent messages) into a JSON file (server_config.json).

Usage: !sync <childID> <realtime> <overindex>

Example: !sync 12345 true 0
Expected Output: A message in the channel: "Server structure has been saved successfully!"

!load

Load & Rename Command

Loads a saved server configuration file and applies an AI-powered renaming process to update category and channel names. The aichat function is used to generate new styled names while preserving the original content.

Usage: !load <configFile>

Example: !load server_config.json
Expected Output: The renaming process is initiated (in a background thread). There may be no immediate message output.

!imports

Import Server Structure

Recreates the server structure from a JSON configuration file. This command creates or updates roles, categories, and channels, applies permission overwrites, and reposts messages via webhooks.

Usage: !imports <configFile>

Example: !imports server_config.json
Expected Output: A message in the channel: "Server structure has been loaded successfully!"

!fullwipe

Full Wipe Command

Deletes all text channels, voice channels, categories, and roles (except for @everyone) in the server. Warning: Use this command with extreme caution.

Example: !fullwipe
Expected Output: The bot deletes the channels, roles, and categories, and then sends "Server wiped!".

!correct

Correction Command

Checks every text channel defined in a configuration file and ensures that every message listed in the file is present. If a message is missing, it is resent via a webhook.

Usage: !correct <configFile>

Example: !correct server_config.json
Expected Output: A message in the channel: "Correction completed successfully!"

!link

Link Management Command

Handles link conversion and posting. It supports several subcommands:

  • link: Converts a Linkvertise URL using bypass and paste services, then posts it via a webhook.
    Usage: !link link <linkvertise_url> <name> [--push]
  • normal: Sends a regular message via webhook.
    Usage: !link normal <message>
  • db: Searches the local database for a name and posts the associated link.
    Usage: !link db <name>
Example (Link Conversion): !link link https://linkvertise.com/abc123 "My Cool Link"
Example (Normal): !link normal This is a test message
Example (DB): !link db MyCoolLink
Expected Output (Link Conversion): A message like @here My Cool Link - [Converted URL] is sent.
Expected Output (Normal): The bot sends the provided message via a webhook.
Expected Output (DB): The bot sends a message with the link associated with "MyCoolLink".

!synchro

Channel Synchronization Command

Aggregates channel IDs from all guilds and logs the list. It can limit the number of channels processed if desired.

Usage: !synchro [channels]

Example: !synchro 5
Expected Output: The bot logs the list of channel IDs to the console and sends a message "Synchro finished!".

!syn

Preset Management Command

Manages channel presets across multiple servers. The subcommands include:

  • presetadd <name>: Creates a new preset with the current channel.
  • presetchan <name or index>: Adds the current channel to an existing preset.
  • presetshow: Displays all available presets.
Example (Add Preset): !syn presetadd MyPreset
Example (Add Channel to Preset): !syn presetchan MyPreset
Example (Show Presets): !syn presetshow
Expected Output: A confirmation message is sent or the list of presets is displayed.

!wipehooks

Webhook Cleanup Command

Deletes all webhooks across all text channels in every guild where the bot is active.

Example: !wipehooks
Expected Output: The bot deletes all webhooks and sends "All webhooks wiped!" in the channel.

!clear

Clear Chat Command

Clears the last 100 messages from the current text channel.

Example: !clear
Expected Output: The bot clears the last 100 messages (output is visible by the change in chat history).

Conclusion

This Discord bot offers extensive functionality for server management, channel synchronization, and automated message handling using webhooks. Its modular design allows for easy customization and integration with external services such as AI renaming and link conversion.

For further customization, refer to the code comments and adjust the helper functions or command logic as needed.
